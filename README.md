MyFirstApp - Многоэкранное Android приложение с SQLite
Автор: Ксения Надбитова  
Курс: Практика ПМ-02 (Мобильная разработка)
О проекте:
Многоэкранное Android-приложение, разработанное в рамках практических занятий. Приложение демонстрирует работу с несколькими Activity, навигацию, создание отдельного модуля для утилит, поддержку многооконного режима и интеграцию с SQLite для CRUD операций.
День 1: Базовое многооконное приложение
Функционал:
- Создание проекта в Android Studio
- Два экрана (MainActivity и ProfileActivity)
- Переход между экранами по кнопке (Intent)
- Обработка событий onClickListener
- Базовая навигация
День 2: ListView и модульность
Функционал:
- Добавление ListView на главный экран
- Обработка кликов по элементам списка (onItemClick)
- Создание новых Activity:
  - CalcActivity - экран для расчётов
  - SettingsActivity - экран настроек
- Создание отдельного модуля Android Library (utils)
- Подключение модуля к приложению
- Класс MathUtils с методами:
  - `add(int a, int b)` - сложение двух чисел
  - `calculateBmi(double weightKg, double heightMeters)` - расчёт ИМТ
- Поддержка многооконного режима** (split screen)
- Анимация переходов между Activity (slide_in_right, slide_out_left)
День 3: SQLite и CRUD операции
Функционал:
1. Создание базы данных SQLite
- Класс `DatabaseHelper` (SQLiteOpenHelper)
- Таблица `tasks` с полями: id, title, description
2. Модель данных
- Класс `Task` с getters и setters
3. CRUD операции
- CREATE - добавление новых задач
- READ - отображение списка задач
- UPDATE - редактирование существующих задач
- DELETE - удаление задач
4. Интеграция с UI
- ListView для отображения задач
- EditText для ввода названия и описания
- Кнопки: Добавить, Обновить, Редактировать, Удалить

- ✅ Сортировка списка: задачи отображаются в алфавитном порядке (ORDER BY title ASC)
